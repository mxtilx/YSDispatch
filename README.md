# YSDispatch
Dispatch server for certain anime game

## Usage

1.Clone the Github Repository

```powershell
git clone https://github.com/tamilpp25/YSDispatch.git
```

2.Extract and open the repository in the Terminal

```powershell
# Assuming that you are in Windows
cd YSDispatch
```

3.Run the following

```powershell
npm install
```

```powershell
npm start
```
